# ARC AF3 Jobs

## ARC layout

All persistent AF3 files live under:

- `/data/chem-proteindesign/sjoh5764/af3_case_study`

Expected subdirectories:

- `/data/chem-proteindesign/sjoh5764/af3_case_study/code`
- `/data/chem-proteindesign/sjoh5764/af3_case_study/models`
- `/data/chem-proteindesign/sjoh5764/af3_case_study/databases`
- `/data/chem-proteindesign/sjoh5764/af3_case_study/input`
- `/data/chem-proteindesign/sjoh5764/af3_case_study/output`
- `/data/chem-proteindesign/sjoh5764/af3_case_study/logs`
- `/data/chem-proteindesign/sjoh5764/af3_case_study/jax_cache`

ARC job scripts live in:

- `/data/chem-proteindesign/sjoh5764/af3_case_study/code/arc_scripts`

## Copy a generated AF3 JSON into `input/`

Example for a full-glycan JSON generated by this repo:

```bash
cp \
  /data/chem-proteindesign/sjoh5764/af3_case_study/code/data/outputs/output_3AVE/top_designs_for_AF3/3AVE_protein_full_sequon_fixed_top1_AF3_full_glycans.json \
  /data/chem-proteindesign/sjoh5764/af3_case_study/input/
```

## Submit the CPU data-pipeline stage

Edit the placeholders at the top of `run_af3_data_pipeline.slurm` first:

- `CONTAINER_PATH`
- `AF3_ENTRYPOINT`
- `DB_DIR` if your database location differs from `/data/chem-proteindesign/sjoh5764/af3_case_study/databases`

Then submit:

```bash
sbatch \
  /data/chem-proteindesign/sjoh5764/af3_case_study/code/arc_scripts/run_af3_data_pipeline.slurm \
  3AVE_protein_full_sequon_fixed_top1_AF3_full_glycans.json
```

This writes the reusable data-pipeline artifact to:

- `/data/chem-proteindesign/sjoh5764/af3_case_study/output/3AVE_full_sequon_fixed_full_glycans/3AVE_full_sequon_fixed_full_glycans_data.json`

That output folder name comes from the AF3 JSON `name` field. For the example
JSON generated by `generate_full_glycan_af3_json.py`, the `name` is
`3AVE_full_sequon_fixed_full_glycans`.

## Submit the GPU inference stage

After the `_data.json` exists, submit:

```bash
sbatch \
  /data/chem-proteindesign/sjoh5764/af3_case_study/code/arc_scripts/run_af3_inference.slurm \
  3AVE_full_sequon_fixed_full_glycans
```

You can also pass the full `_data.json` path instead of the job name.

## Import and validate afterward

Run the import/validation steps from an interactive session rather than on the
login node:

```bash
srun -p interactive --time=01:00:00 --cpus-per-task=4 --mem=16G --pty /bin/bash
cd /data/chem-proteindesign/sjoh5764/af3_case_study/code

python 05_af3_validation/organize_af3_results.py \
  /data/chem-proteindesign/sjoh5764/af3_case_study/output

python 05_af3_validation/analyze_af3_validation.py
python 05_af3_validation/analyze_local_glycosites.py
```
